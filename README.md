# 概要
windowsでvoicevoxを使って通知を読み上げてもらうアプリです。

# 設定ファイル

C:\Users\（ユーザー名）\AppData\Local\Packages\8451988f-bf41-4f5d-9f30-926569867ce4_wrfn4kgh3xyrw\LocalStateにconfig.jsonを作ってもらって、キーをvoicevox_engine_Path,値をvoicevox_engineの実行ファイルrun.exeのパスにしてもらうと動きます。

# クレジット

本ソフトウェアは [VOICEVOX](https://voicevox.hiroshiba.jp/) を利用しています。
